# sleep-EOG-analysis
Preprocessing EOG signals and modeling for automated sleep staging

# data
Haaglanden Medisch Centrum sleep staging database
URL: https://physionet.org/content/hmc-sleep-staging/1.1

# model
2 layers of 1D-CNN + LSTM
